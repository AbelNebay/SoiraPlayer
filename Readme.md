# Soira Player – DRM Video Player

## Overview
Soira Player is a secure DRM-based video player designed to stream and protect premium video content from unauthorized access or downloads.

## Features
- DRM-protected playback
- Secure streaming (anti-download)
- Offline encrypted playback
- Multi-format video support
- Lightweight and user-friendly UI

## Tech Stack
- Java (Android)
- ExoPlayer
- DRM encryption
- PHP backend (optional)

## Use Case
- E-learning platforms
- Paid video content distribution
- Secure media delivery
